# Lotto
Lotto Simulator

## How to Run

1. Ensure you have Python and Flask installed in your environment (e.g., conda activate lotto-env).
2. Run `python app.py` in the Lotto directory.
3. Open your browser and go to http://127.0.0.1:5000 to access the web app.

## Features

- Input 5 unique numbers between 1 and 50.
- Simulates a lotto draw with 5 random numbers.
- Checks for matches: 3 matches = 1M Euros, 4 = 20M, 5 = 200M.
- Keeps a history of games in the session.
